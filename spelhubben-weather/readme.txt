Lightweight weather widget/block with optional multi-provider consensus (Open-Meteo, SMHI, Yr/MET Norway). Shortcode + block + classic widget. Optional Leaflet map.

== Description ==
This plugin displays current weather and an optional forecast. It can aggregate data from northern-Europe friendly providers (Open-Meteo, SMHI, and Yr/MET Norway) and compute a simple consensus. It includes:
- Gutenberg block and shortcode
- Optional Leaflet map
- Daily forecast and multiple layouts
- Fully translatable

*Not affiliated with Open-Meteo, SMHI, Yr/MET Norway, Leaflet, or OpenStreetMap. Names are used for descriptive purposes only. Map data © OpenStreetMap contributors, ODbL.*

== Installation ==
1. Upload the plugin and activate it.
2. Add the block “Spelhubben Weather” or use the `[spelhubben_weather]` shortcode.

== Frequently Asked Questions ==
= Where does the data come from? =
From public APIs such as Open-Meteo, SMHI and Yr/MET Norway (depending on your settings).
